Expand description

Determines the strategy to fetch a storage from the storage map. The two implementors from write_full_isotope_storage and write_partial_isotope_storage would either lazily create or panic on missing storage.

This trait is not symmetric to StorageMapProcessorRef because the equivalent API may lead to Polonius compile errors.
