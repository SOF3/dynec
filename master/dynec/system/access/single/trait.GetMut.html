fn try_get_mut( &mut self, entity: impl Ref<Archetype = Self::Arch> ) -> Option<&mut Self::Comp>

Returns a mutable reference to the component for the specified entity, or None if the component is not present in the entity.

Note that this function returns Option<&mut C>, not &mut Option<C>. This means setting the Option itself to Some/None will not modify any stored value. Use set to add/remove a component.

source

fn try_get_many_mut<const N: usize>( &mut self, entities: [impl Ref<Archetype = Self::Arch>; N] ) -> Option<[&mut Self::Comp; N]>

Returns mutable references to the components for the specified entities.

Returns None if any component is not present in the entity or if the same entity is passed multiple times.

Iterates over mutable references to all initialized components in this storage.
