Manages sharded entity ID allocation and deallocation.

fn shards<U, F: Fn(Self::Shard) -> U>(&mut self, vec: &mut Vec<U>, f: F)

Returns a vector of shards where each shard references internal states in the allocator.

The length of the result must be num_shards in Ealloc::new. The caller shall shuffle the results returned by this method.

source

fn allocate(&mut self, hint: Self::AllocHint) -> Self::Raw

Allocate an ID. Can only be used between out-of-cycle.

source

Deallocates the IDs, allowing them to be reused in future calls to Shard::allocate.
