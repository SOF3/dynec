source

fn get( &mut self, entity: impl Ref<Archetype = Self::Arch>, discrim: Self::Key ) -> &Self::Comp
where Self::Comp: Must<Self::Arch>,

Retrieves the component for the given entity and discriminant.

This method is infallible for correctly implemented comp::Must, which returns the auto-initialized value for missing components.

source

fn try_get( &mut self, entity: impl Ref<Archetype = Self::Arch>, key: Self::Key ) -> Option<&Self::Comp>

Returns an immutable reference to the component for the specified entity and discriminant, or the default value for isotopes with a default initializer or None if the component is not present in the entity.

source

fn known_discrims<'t>(&'t self) -> Self::KnownDiscrims<'t>

Iterates over all known discriminants of the component type.

The yielded discriminants are not in any guaranteed order.

source

fn get_all<'t, E: Ref<Archetype = Self::Arch>>( &'t self, entity: E ) -> Self::GetAll<'t, E>

Iterates over all known isotopes for a specific entity.

source

fn iter<'t>(&'t mut self, key: Self::Key) -> Self::Iter<'t>

Iterates over all components of a specific discriminant.

Note that the initializer is not called for lazy-initialized isotope components. To avoid confusing behavior, do not use this function if C: comp::Must<A>.

source

fn split<'t, const N: usize>( &'t mut self, keys: [Self::Key; N] ) -> [AccessSingle<Self::Arch, Self::Comp, Self::Split<'t, N>>; N]

Splits the accessor into multiple immutable AccessSingle accessors so that they can be used independently.
