Expand description

Provides access to an isotope component in a specific archetype.

DiscrimSetKey is the type used to index the discriminant, depending on the type of discriminant set given. For vec and array discriminant sets, this should be usize. For partial isotope access, K is usually usize. For full isotope access, K is the discriminant type.

To share the same API as ReadIsotopeFull, immutable getters still require &mut self, but there are *_ref variants for these functions that just require &self.
